Basic PDF Encodings
Standard Adobe standard Latin-text encoding
WinAnsi (default) Windows Code Page 1252, often called the “Windows ANSI” encoding. This is the standard Windows encoding for Latin text in Western writing systems
MacExpert An encoding for use with expert fonts—ones containing the expert character set. PDF has a predefined encoding named MacExpertEncoding. Despite its name, it is not a platform-specific encoding; however, only certain fonts have the appropriate character set for use with this encoding
PdfDoc It is not customary to use this encoding to show text from fonts
MacRoman Mac OS standard encoding for Latin text in Western writing systems
Western 8-bit single-byte coded graphic character sets
ISO8859-1 Latin-1
Afrikaans, Albanian, Basque, Breton, Catalan, Danish, English (UK and US), Faroese, Galician, German, Icelandic, Irish (new orthography), Italian, Latin (basic classical orthography), Leonese, Luxembourgish (basic classical orthography), Norwegian (Bokmål and Nynorsk), Occitan, Portuguese, Rhaeto-Romanic, Scottish Gaelic, Spanish, Swahili, Swedish, Walloon
ISO8859-2 Latin-2
Bosnian, Croatian, Czech, German (fully compatible with ISO8859-1 for German texts), Hungarian, Polish, Serbian Latin, Slovak, Slovene, Upper Sorbia, Lower Sorbian
ISO8859-4 Latin-4 or North European
Estonian, Latvian, Lithuanian, Greenlandic, and Sami
ISO8859-5 Latin-5
Bulgarian, Belarusian, Russian, Serbian, Macedonian AND Ukrainian
ISO8859-6 to ISO8859-16 Other Latin Encoding Schemes
CP1250 Central European and Eastern European languages
Polish, Czech, Slovak, Hungarian, Slovene, Bosnian, Croatian, Serbian (Latin script), Romanian and Albanian
CP1251 to CP1258 Other encoding schemes

Identity−H

The horizontal identity mapping for 2-byte CIDs; may be used with CIDFonts using any Registry, Ordering, and Supplement values. It maps 2-byte character codes ranging from 0 to 65,535 to the same 2-byte CID value, interpreted high-order byte first

Identity−V

Vertical version of Identity−H. The mapping is the same as for Identity−H.
