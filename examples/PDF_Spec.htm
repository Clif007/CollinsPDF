RobotTalk Commands:
  • OPEN [[as=PDF | STRING | BASE64 | DataURL ]  [Target=name] [Data=html-string]
  • WRITE  file-specification | url-specification [as=PDF | Base64 | DataURL] [Data=html-string]
  • ADD [Data=html-String]
  • RETURN as=STRING | BASE64 | DataURL | OBJECT_PDF | OBJECT_HTML | ERROR_MESSAGES

The html source string can be passed using the "Data" qualifier, as a file name, or as the second argument of "Adam"
